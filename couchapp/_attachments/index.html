Couch View API

  • Query (view function) /[db_name]/_design/[design_doc_name]/_view/[view_name]
  • Execute a show function /[db_name]/_design/[design_doc_name]/_show/[function_name]/[doc_id]
  • Execute a list function /[db_name]/_design/[design_doc_name]/_list/[function_name]/[view_name]
